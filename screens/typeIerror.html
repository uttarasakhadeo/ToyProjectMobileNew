Type I Error

A Type I error occurs when a scientist declares a difference when none is actually present. The Type I error rate is the probability of a Type I error occurring. Type I error rates range from 0 to 1. The most commonly used values are 0.01, 0.05, and 0.1.
Enter each Type I error value into the text box and click enter. You may enter up to 5 values.
